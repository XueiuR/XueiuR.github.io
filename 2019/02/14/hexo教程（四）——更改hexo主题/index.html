hexo教程（四）——更改hexo主题

发表于 2019-02-14
字数统计: 521 | 阅读时长 ≈ 1

在之前的内容中，我们成功地发布了第一篇博客，这次，我们看一下如何更改hexo主题

更改hexo主题

认识 _config.yml 文件

在Blog/hexo/文件夹中，有一个_config.yml文件，里面存放了博客网站的配置信息，我们用文本编辑器打开它。打开以后，我们挑需要的内容来看。

  • 网站

title 网站的标题

subtitle 网站的副标题

description 网站的描述

author 网站作者的名字

language 网站的语言

timezone 网站的时区。默认使用本地电脑的时区

  • 分页

per_page 每页显示的文章数量。如果设置为0，则表示不分页

pagination_dir 分页的目录

  • 扩展

theme 主题。如果设置为false，则表示禁用主题。

【注意】在编辑_config.yml文件时，一定要在:和输入的内容之间加一个空格

挑选喜欢的网页主题

  • 在Hexo主题中选择我们喜欢的主题，这里我选择的是Anatole主题。

  • 点击此处下载主题压缩包，将其解压到Blog/hexo/themes中，将文件夹名由hexo-theme-Anatole-1.1.0改为anatole

  • 将_config.yml文件中上文提到的theme:改为theme: anatole

  • 在命令行中输入npm install --save hexo-render-pug hexo-generator-archive hexo-generator-tag hexo-generator-index hexo-generator-category，安装和配置必要的插件,大概一分钟左右。

  • 在命令行里输入npm install --save hexo-render-pug

【注意】一定要注意，这些命令语句是在hexo的根目录下输入的，而不是主题anatole的目录下

  • 我们可以在_config.yml文件中，添加avatar: ，后面附带一个图片的链接，就可以作为自己博客的头像了

  • 在_config.yml文件中,添加github: ，后面附带上github用户名，就可以在主页上出现你的github链接

将设置应用到博客网站

我们执行和上传博文一样的方式,执行hexo g和hexo d命令，再打开你的博客网站，就会发现网站已经成功更改了主题
